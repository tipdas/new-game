Описание

Игрок оказывается в лабиринте и должен пройти через несколько локаций, решая головоломки для открытия новых областей.

В игре реализованы:
    •    Несколько локаций
    •    ASCII-карты локаций
    •    Система переходов между локациями
    •    Головоломки разных типов
    •    Открытие новых локаций после решения загадок

    Запуск

Скомпилируйте проект и запустите полученный исполняемый файл.

Пример для g++:

g++ main.cpp -o Ugra

После компиляции:

./Ugra
